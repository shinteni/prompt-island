Local-first

Local by default.

Vibelsland Free reads local Claude Code, Codex CLI, and Codex Desktop state only to render the island and handle approvals. It creates no account, uploads no telemetry, and does not sync session content to a remote server.

Data boundary

What it reads and writes.

It reads

  • Local Claude Code, Codex CLI, and Codex Desktop state.
  • Local status summaries used to show activity, tool calls, and approvals.
  • Claude and Codex local connection settings and runtime state.

It writes

  • App config: ~/Library/Application Support/VibelslandFree/config.json.
  • App logs: ~/Library/Logs/VibelslandFree/app.log.
  • Local connection runtime files: ~/.vibelsland-free.

Retention and deletion

Local logs and removal paths are explicit.

Local connection files are used only for communication on your Mac. Logs are for troubleshooting and are not uploaded by the app. After disconnecting local integrations, runtime files, config, and logs can be deleted manually.

Local connection files

  • Runtime files live in ~/.vibelsland-free and let Vibelsland Free communicate with tools on your Mac.
  • These files are used only to confirm local request sources. They are not online account credentials.
  • Deleting ~/.vibelsland-free removes runtime files; the app or connection entries recreate the required files on the next launch.

Logs and deletion

  • Logs live at ~/Library/Logs/VibelslandFree/app.log and record connection state, event types, timestamps, local paths, and error reasons.
  • The app does not upload logs. Before sharing logs, remove account details, private file paths, conversation content, and other sensitive information.
  • When you stop using the app, you can delete ~/Library/Application Support/VibelslandFree, ~/Library/Logs/VibelslandFree, and ~/.vibelsland-free.

Control

Local connections change only when you choose.

The app only changes corresponding configuration when you install, repair, or uninstall local connections. Status sent to the app is summarized so it keeps only what is needed to show current activity.

Network

Core features do not require internet access.

Vibelsland Free status display, source controls, and local approvals work on your Mac. Claude Code, Codex CLI, and Codex Desktop may have their own network connections independently of this app.

Vibelsland Free is an independent project and is not affiliated with Anthropic, OpenAI, Claude, or Codex.

Website privacy

The website boundary is explicit too.

This static website sets no first-party cookies, embeds no analytics script, and collects no forms. Pages are hosted by GitHub Pages; GitHub Release, source, and Issues links move you into GitHub's service boundary.

This site does not collect

  • No first-party account, newsletter, contact form, or tracking script.
  • No website visit data is synced back to Vibelsland Free.
  • Download page visits are not linked to local app state.

External service boundary

  • GitHub Pages, GitHub Releases, and GitHub Issues are provided by GitHub.
  • GitHub terms and privacy policy apply when you download, inspect source, or open an issue.
  • Do not paste account details, access credentials, conversation content, or unredacted logs into public issues.
