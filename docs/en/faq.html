Install and trust

What to confirm before download.

Is Vibelsland Free free?

Yes. v0.1.0 is free to download, source-available, and distributed through GitHub Releases.

Why does first launch require an extra step?

v0.1.0 uses ad-hoc signing, so macOS may require an extra confirmation on first launch. Control-click the app and choose Open.

How do I verify the download?

Put the zip and matching .sha256 file in one folder, then run shasum -a 256 -c.

Scope

What it reads and what it supports.

Which AI coding tools are supported?

v0.1.0 supports local status display, tool-call summaries, and approval handling for Claude Code, Codex CLI, and Codex Desktop.

Do core features require internet access?

No. Status display, source controls, and approval bridging run on your Mac.

How do I stop or uninstall hooks?

Uninstall hooks by source in Settings. The app changes configuration only after your action.
