Forms

One island. Three forms.

The island morphs with your session. Color is the protocol, and the position stays out of your way.

01

Running

An RGB ring marks the active tool, so you know a task is still moving without switching back to the terminal.

02

Approval

When permission is needed, the island goes amber. Supported sources can handle accept once, accept for session, decline, and cancel task; Codex CLI hooks return only allow or deny.

03

Many

When multiple agents run at once, stacked avatars show who is thinking and who is waiting.

Install

Three minutes. You are running.

v0.2.0 ships with ad-hoc signing, so first launch needs manual trust in System Settings.

  1. Download and unzipGrab the zip from GitHub Releases and drag >_ - island.app into Applications.
  2. Verify SHA-256Use shasum -a 256 -c to check the official checksum file against the download.
  3. Launch and install HooksIf macOS says Apple cannot verify the app, click Done, allow it from Privacy & Security, then install Hooks.
$ shasum -a 256 -c Vibelsland-Free-0.2.0-macos.zip.sha256
