Install

Install in three steps.

  1. 01 Download the zip

    Download Vibelsland-Free-0.1.0-macos.zip from GitHub Releases.

  2. 02 Move to Applications

    Unzip it, drag the app into Applications, then launch it like a normal macOS app.

  3. 03 Enable local sources

    Open Settings and enable the Claude Code, Codex CLI, and Codex Desktop sources you need.

Signing and checksum

What to know before installing.

v0.1.0 uses ad-hoc signing and is not yet Developer ID signed or Apple notarized. If macOS blocks first launch, right-click and choose Open. The download includes a SHA-256 checksum and the source is available on GitHub.
