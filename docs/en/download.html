Install

Install in four steps.

  1. 01 Download zip

    Download the zip and matching .sha256 file into the same folder, such as Downloads.

  2. 02 Verify SHA-256

    Run cd ~/Downloads && shasum -a 256 -c Vibelsland-Free-0.2.0-macos.zip.sha256.

  3. 03 Unzip and launch

    Move the app to Applications and open it once. If macOS says Apple cannot verify the app, click Done. Do not click Move to Trash.

  4. 04 Open Anyway in System Settings

    Go to System Settings → Privacy & Security, click Open Anyway, confirm, then open the app again.
