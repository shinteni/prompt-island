Install

Install in three steps.

    Download Vibelsland-Free-0.1.0-macos.zip from GitHub Releases.

  2. 02 Move to Applications

    Unzip it, drag the app into Applications, then follow the first-launch note below.

  3. 03 Enable local sources

    Open Settings and enable the Claude Code, Codex CLI, and Codex Desktop sources you need.

Signing and checksum

What to know before installing.

v0.1.0 uses ad-hoc signing, so macOS may require an extra confirmation on first launch. Control-click the app and choose Open. The download includes a SHA-256 checksum and the source is available on GitHub.
