インストールと信頼

ダウンロード前に確認すること。

Vibelsland Free は無料ですか？

はい。v0.1.0 は無料でダウンロードでき、ソースも公開されています。配布は GitHub Releases です。

初回起動時に追加確認が必要なのはなぜですか？

v0.1.0 は ad-hoc 署名を採用しているため、初回起動時に macOS が追加確認を求める場合があります。Control キーを押しながらアプリをクリックし、「開く（Open）」を選んでください。

ダウンロードをどう検証しますか？

zip と同名の .sha256 ファイルを同じフォルダに置き、shasum -a 256 -c Vibelsland-Free-0.1.0-macos.zip.sha256 を実行します。

対応範囲

読み取るもの、対応するもの。

どの AI コーディングツールに対応していますか？

v0.1.0 は Claude Code、Codex CLI、Codex Desktop のローカル状態表示、ツール実行サマリー、承認処理に対応しています。

主要機能にインターネット接続は必要ですか？

必要ありません。状態表示、ソース制御、承認ブリッジは Mac 内で動作します。

Hooks を停止または削除できますか？

設定画面でソースごとに hooks を削除できます。アプリは操作したときだけ設定を変更します。
