インストールと信頼

ダウンロード前に確認すること。

Vibelsland Free は無料ですか？

はい。v0.1.0 は無料でダウンロードでき、ソースも公開されています。配布は GitHub Releases です。

初回起動時に追加確認が必要なのはなぜですか？

v0.1.0 は ad-hoc 署名を採用しているため、初回起動時に macOS が追加確認を求める場合があります。Control キーを押しながらアプリをクリックし、「開く（Open）」を選んでください。

Developer ID notarization がないのはなぜですか？

v0.1.0 は ad-hoc 署名で、Developer ID notarization ではありません。Gatekeeper が Control-click Open を求める場合があります。現在のパッケージでは spctl の rejected または not accepted は想定される結果です。SHA-256 は GitHub Release の配布物と一致することを確認するもので、開発者本人性を証明しません。

Intel Mac で使えますか？

v0.1.0 の配布パッケージは Apple Silicon / arm64 と macOS 14+ 向けです。Intel Mac は現在のサポート範囲に含まれません。

ダウンロードをどう検証しますか？

zip と同名の .sha256 ファイルを同じフォルダに置き、shasum -a 256 -c Vibelsland-Free-0.1.0-macos.zip.sha256 を実行します。

対応範囲

読み取るもの、対応するもの。

どの AI コーディングツールに対応していますか？

v0.1.0 は Claude Code、Codex CLI、Codex Desktop のローカル状態表示、ツール実行サマリー、承認処理に対応しています。

Anthropic や OpenAI に内容を送信しますか？

Vibelsland Free 自体はアカウントを作らず、テレメトリを送らず、セッション内容を自前のサーバーへ同期しません。Claude Code、Codex CLI、Codex Desktop はそれぞれ独自にネットワーク接続を使う場合があります。

Anthropic、OpenAI、Claude、Codex との関係は？

Vibelsland Free は独立したツールです。Anthropic、OpenAI、Claude、Codex に所属せず、代表せず、推奨を受けたものでもありません。製品名はローカル互換性を説明するためだけに使っています。

app.log に会話内容は保存されますか？

app.log は主に接続状態、イベント種別、時刻、ローカルパス、エラー理由を記録します。完全な会話をログ本文として保存するものではありません。状態表示では対応ツールから Mac 内の最新状態サマリーを読み取る場合があります。ログ共有前に、アカウント情報、個人的なファイルパス、会話内容、その他の機密情報を隠してください。

ダウンロード後のアプリ名が「>_ - island.app」なのはなぜですか？

これは macOS のアプリ bundle 名です。Web サイト、リリースパッケージ、ソースでは引き続き Vibelsland Free を製品名として使います。この名前は Anthropic、OpenAI、Claude、Codex との公式関係を示すものではありません。

ローカル接続を停止または削除できますか？

Uninstall Hooks は、ローカル状態表示のために Vibelsland Free が追加した接続項目を削除します。アプリ本体、ログ、設定、~/.vibelsland-free、第三者ツールが保存した記録は自動削除しません。
