ローカルファースト

標準で Mac 内に。

Vibelsland Free は Claude Code、Codex CLI、Codex Desktop の Mac 内の状態だけを読み取り、アイランド表示と承認処理に使います。アカウントを作らず、テレメトリを送らず、セッション内容をリモートサーバーへ同期しません。

インストールと信頼の説明を見る ソース内のプライバシーファイルを見る
アカウント 不要
テレメトリ 送信しない
同期 クラウド非依存

データ境界

読み取るもの、書き込むもの。

読み取るもの

  • Claude Code、Codex CLI、Codex Desktop の Mac 内の状態。
  • 活動状態、ツール実行、token サマリー、承認を表示するためのローカルセッション情報。
  • Claude と Codex のローカル hook 設定、セッションファイルのパス、実行状態。

書き込むもの

  • アプリ設定: ~/Library/Application Support/VibelslandFree/config.json。
  • アプリログ: ~/Library/Logs/VibelslandFree/app.log。
  • ローカルブリッジのランタイムファイル: ~/.vibelsland-free。

保持と削除

ローカル token、ログ、削除場所を明確にします。

ローカル bridge token は Mac 内の socket 認証だけに使われます。ログはトラブルシューティング用で、アプリがアップロードすることはありません。hooks をアンインストールした後、実行ファイル、設定、ログは手動で削除できます。

ローカル bridge token

  • 実行ファイルは ~/.vibelsland-free にあり、ローカル Unix socket とローカル bridge token を含みます。
  • token は Mac 内のプロセス間でリクエスト元を確認するためだけに使われ、オンラインアカウントの認証情報ではありません。
  • ~/.vibelsland-free を削除すると実行 token は消えます。次回起動時にアプリまたは hooks が必要な実行ファイルを作り直します。

ログと削除

  • ログは ~/Library/Logs/VibelslandFree/app.log にあり、接続状態、イベント種別、時刻、ローカルパス、エラー理由を記録します。
  • アプリはログをアップロードしません。ログを共有する前に、機密性のあるパス、プロンプト、token、セッション内容を削除してください。
  • 使用をやめる場合は、~/Library/Application Support/VibelslandFree、~/Library/Logs/VibelslandFree、~/.vibelsland-free を削除できます。

ユーザー制御

Hooks は選択したときだけ変更します。

アプリは hooks のインストール、修復、アンインストール時だけ設定を変更します。Hook payload は先にフィルタリングされ、セッション ID、イベント種別、ワークスペース、承認 ID、ツール名、パス、コマンドなど必要なメタデータだけを保持します。

ネットワーク境界

主要機能にインターネット接続は不要です。

Vibelsland Free のステータス表示、ソース制御、ローカル承認ブリッジは Mac 内で動作します。Claude Code、Codex CLI、Codex Desktop はそれぞれ独自にネットワーク接続を使う場合があります。

サイトのプライバシー

Web サイト側の境界も明確にします。

この静的サイトは独自の cookies を設定せず、分析スクリプトやフォームも置いていません。ページは GitHub Pages でホストされ、GitHub Release、ソース、Issues へのリンクを開くと GitHub のサービス境界に移ります。

このサイトが収集しないもの

  • 独自アカウント、ニュースレター、問い合わせフォーム、追跡スクリプトはありません。
  • サイト訪問データを Vibelsland Free に同期しません。
  • ダウンロードページの閲覧とローカルアプリ状態を関連付けません。

外部サービス境界

  • GitHub Pages、GitHub Releases、GitHub Issues は GitHub が提供します。
  • ダウンロード、ソース閲覧、issue 作成時は GitHub の規約とプライバシーポリシーが適用されます。
  • 公開 issue に token、prompt、セッション内容、未編集ログを貼らないでください。
