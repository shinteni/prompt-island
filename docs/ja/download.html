インストール

三つの手順でインストール。

  1. 01 zip をダウンロード

    GitHub Releases から Vibelsland-Free-0.1.0-macos.zip をダウンロードします。

  2. 02 Applications へ移動

    解凍して >_ - island.app を Applications に移動し、通常の macOS アプリとして起動します。

  3. 03 ローカルソースを有効化

    設定画面で必要な Claude Code、Codex CLI、Codex Desktop のソースを有効化します。

署名とチェックサム

インストール前に知っておくこと。

v0.1.0 は ad-hoc 署名で、Developer ID 署名と Apple notarization にはまだ対応していません。初回起動時に macOS が止めた場合は、右クリックして「開く」を選んでください。ダウンロードには SHA-256 チェックサムがあり、ソースは GitHub で確認できます。
