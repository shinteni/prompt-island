v0.1.0

最初の公開版。

このリリースは Claude Code、Codex CLI、Codex Desktop のローカル状態を、低干渉な macOS アイランドにまとめます。長いタスク、ツール実行、承認が頻繁に発生する AI コーディング作業で、現在のウィンドウを保ったまま状態を確認できます。

主要機能

v0.1.0 に含まれる内容。

画面上部のステータスアイランド

  • AI コーディングの活動状態、最近のツール実行、token サマリーを macOS 上部に表示します。
  • アイドル時は小さく、長い処理や重要な状態では展開します。
  • Claude Code、Codex CLI、Codex Desktop の並行ワークフロー向けです。

ローカル承認処理

  • 許可、拒否、続行、キャンセルを macOS ネイティブの画面で処理します。
  • 承認 ID、ツール名、パス、コマンドなどの有用な文脈を保持します。
  • ターミナル、デスクトップクライアント、ログ間の切り替えを減らします。

ソースと hook の制御

  • Claude Code、Codex CLI、Codex Desktop のソースを個別に有効化できます。
  • 設定画面からローカル hooks をインストール、修復、アンインストールできます。
  • ブリッジの実行ファイルとログは Mac 内に残り、排障と整理に使えます。

ローカルファースト境界

  • このアプリはアカウント作成、テレメトリ送信、リモートセッション同期を行いません。
  • 主要な状態表示、ソース制御、承認ブリッジは Mac 内で動作します。
  • プライバシーページで読み取り、書き込み、hook payload、ネットワーク境界を説明しています。

既知の境界

v0.1.0 の製品範囲。

v0.1.0 は macOS 14+ 上のローカル状態表示と承認ブリッジに焦点を当てています。Claude Code、Codex CLI、Codex Desktop はそれぞれのアカウント、ネットワーク、セッション動作を保ちます。Vibelsland Free は Mac 上で利用できるローカル状態を整理し、ソースごとに表示を制御できるようにします。
