v0.2.0 の新機能

ワンクリック更新、承認キュー、より磨かれたデスクトップ体験。

docs/release.json は GitHub Releases に公開中の v0.2.0 パッケージを記録し、ダウンロード、チェックサム、Homebrew cask は同じメタデータから生成されます。

更新と配布

  • 設定画面の「確認して更新」ワンクリックで、ダウンロード、SHA-256 検証、置き換え、再起動まで完了。
  • パッケージは Apple Silicon と Intel 両対応の Universal になり、リポジトリは Homebrew tap を兼ねます。

承認体験

  • 複数の承認は待ち時間順のキューで表示し、行ごとに許可/拒否できます。
  • 許可/拒否ボタン付きの通知センター連携（任意）と、グローバルショートカット ⌃⌥I / ⌃⌥A。

使用量と統計

  • Claude Code セッションに今回/累計トークンと API 換算コストの目安を表示。
  • ローカルのみに保存される「今日 / 直近 7 日」の統計カードを設定画面に追加。いつでも消去できます。

パフォーマンスと手触り

  • 展開/折りたたみと起動アニメーションはディスプレイのリフレッシュレートに追従する GPU 描画になり、ホバー/押下フィードバックを全面追加。
  • バックグラウンドのタイマーは必要時のみ起動し、アイドル時の消費を削減。システムの「視差効果を減らす」にも完全対応。

v0.1.0

最初の公開版。

このリリースは Claude Code、Codex CLI、Codex Desktop のローカル状態を、低干渉な macOS アイランドにまとめます。長いタスク、ツール実行、承認が頻繁に発生する AI コーディング作業で、現在のウィンドウを保ったまま状態を確認できます。

主要機能

v0.1.0 に含まれる内容。

画面上部のステータスアイランド

  • AI コーディングの活動状態、最近のツール実行、状態サマリーを macOS 上部に表示します。
  • アイドル時は小さく、長い処理や重要な状態では展開します。
  • Claude Code、Codex CLI、Codex Desktop の並行ワークフロー向けです。

ローカル承認処理

  • 一回許可、セッション中の許可（対応時）、拒否、タスク取消を macOS ネイティブの画面で処理します。Codex CLI Hook の承認は許可または拒否のみです。
  • 承認処理に必要な基本的な文脈を保持します。
  • ターミナル、デスクトップクライアント、ログ間の切り替えを減らします。

ソースとローカル接続

  • Claude Code、Codex CLI、Codex Desktop のソースを個別に有効化できます。
  • 設定画面からローカル接続をインストール、修復、アンインストールできます。
  • 接続の実行ファイルとログは Mac 内に残り、排障と整理に使えます。

ローカルファースト境界

  • このアプリはアカウント作成、テレメトリ送信、リモートセッション同期を行いません。
  • 主要な状態表示、ソース制御、承認処理は Mac 内で動作します。
  • プライバシーページで読み取り、書き込み、ネットワーク境界を説明しています。

既知の境界

v0.1.0 の製品範囲。

v0.1.0 は macOS 14+ 上のローカル状態表示と承認処理に焦点を当てています。Claude Code、Codex CLI、Codex Desktop はそれぞれのアカウント、ネットワーク、セッション動作を保ちます。Vibelsland Free は Mac 上で利用できるローカル状態を整理し、ソースごとに表示を制御できるようにします。
