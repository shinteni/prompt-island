フォーム

一つの島、三つの状態。

セッションに合わせて島は姿を変えます。色がプロトコルで、位置は作業を邪魔しません。

01

実行中

RGB リングが動作中のツールを示し、ターミナルへ戻らなくても進行中だと分かります。

02

承認

権限が必要なときはアンバーに変わり、許可、拒否、続行、キャンセルを一箇所に集めます。

03

複数

複数の agent が同時に動くとき、重なったアバターで考え中と待機中を見せます。

ローカル優先

アップロードなし。同期なし。アカウントなし。

Mac 上のローカル状態ファイルを読み取り、島に描画するだけ。クラウドなし、テレメトリなし、ソースは完全公開。

0バイトのテレメトリが Mac を離れます
0個のアカウントやサインインが必要
100%ローカル、オープンソース、監査可能

インストール

三分で走り出せます。

v0.1.0 は ad-hoc 署名のため、初回起動時にシステム設定での手動承認が必要です。

  1. ダウンロードして解凍GitHub Releases から zip を取得して >_ - island.app をアプリケーションフォルダへ移動します。
  2. SHA-256 を検証shasum -a 256 -c で公式チェックサムと突き合わせ、改ざんがないことを確認します。
  3. 起動して Hooks をインストールGatekeeper の確認後にアプリを開き、メニューバーまたは設定から Hooks をインストールします。
