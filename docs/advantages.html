产品优势

状态可见，工作不断。

Vibelsland Free 把会话进度、工具调用和审批请求整理成可扫视的状态显示。它停留在屏幕顶部，让你继续写代码，同时保留关键上下文。

下载 macOS 版 查看本地数据边界

价值

AI 编程时最容易漏看的三件事。

01

长任务是否还在推进

构建、搜索和工具调用常常持续数分钟。浮岛把最近状态留在屏幕顶部，不需要反复切回终端。

02

审批请求在哪里出现

允许一次、本轮允许（支持时）、拒绝和取消任务集中处理；Codex CLI Hook 审批只回传允许或拒绝。

03

多个 AI 工具谁在活动

Claude Code、Codex CLI 和 Codex Desktop 的状态统一呈现，更适合多会话并行工作。

定位

把状态可见性接入你的现有工作流。

没有浮岛时

  • 状态散在终端、桌面客户端和日志里。
  • 长任务只能靠不断切窗口判断。
  • 审批请求出现时容易漏掉上下文。

使用 Vibelsland Free

  • 顶部保留一个低干扰状态入口。
  • 任务、工具调用和审批请求统一呈现。
  • 空闲时收起，关键状态出现时展开。

macOS 原生

像系统状态一样出现。

浮岛、设置页、审批窗口和来源开关都围绕 macOS 的使用节奏设计。它可以长期常驻，也能在你不需要时安静隐藏。

形态 小圆点 / 药丸 / 面板
位置 屏幕顶部常驻
控制 按来源启用或关闭
