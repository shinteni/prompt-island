安装

四步安装。

  1. 01 下载 zip

    把 zip 和同名 .sha256 文件下载到同一目录，例如 Downloads。

  2. 02 校验 SHA-256

    运行 cd ~/Downloads && shasum -a 256 -c Vibelsland-Free-0.2.1-macos.zip.sha256。

  3. 03 解压并启动

    把应用拖入 Applications 后先打开一次；如果出现 Apple 无法验证的提示，请点“完成”，不要点“移到废纸篓”。

  4. 04 在系统设置里仍要打开

    进入“系统设置 → 隐私与安全性”，点击“仍要打开”，确认后再次打开应用。

继续查看完整安装与信任说明 查看本地数据边界

签名与校验

校验下载。

当前包使用 ad-hoc 签名，不是 Developer ID notarization。SHA-256 只确认文件与 GitHub Release 资产一致，不证明开发者身份；首次打开需要在 macOS 隐私与安全性里手动允许。
