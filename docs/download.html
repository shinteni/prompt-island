安装

三步完成安装。

  1. 01 下载压缩包

    从 GitHub Releases 下载 Vibelsland-Free-0.1.0-macos.zip。

  2. 02 拖入 Applications

    解压后把应用拖到 Applications。安装后的应用名显示为 >_ - island.app，请按首次打开说明启动。

  3. 03 启用本地来源

    打开设置页，按需启用 Claude Code、Codex CLI 和 Codex Desktop 的本地状态来源。

签名与校验

安装前需要知道的事项。

v0.1.0 采用 ad-hoc 签名，因此首次打开时 macOS 可能要求额外确认。请按住 Control 点按应用并选择“打开（Open）”。下载包提供 SHA-256 校验，源码可在 GitHub 查看。
