主要功能

v0.1.0 包含的核心能力。

顶部浮岛状态显示

  • 把 AI 编程会话的活动状态、最近工具调用和 token 摘要保持在屏幕顶部。
  • 空闲时收起，长任务运行和关键状态出现时展开。
  • 面向 Claude Code、Codex CLI 和 Codex Desktop 的并行工作流。

本地审批处理

  • 在 macOS 原生界面中处理允许、拒绝、继续和取消。
  • 保留审批 ID、工具名、路径和命令等必要上下文。
  • 减少在终端、桌面客户端和日志之间切换的频率。

来源与 hook 控制

  • 按来源启用 Claude Code、Codex CLI 和 Codex Desktop 状态。
  • 通过设置页安装、修复或卸载本地 hooks。
  • 桥接运行文件和日志保留在本机，便于排障和清理。

本地优先边界

  • 无需创建账号，也不上传遥测或远程同步会话内容。
  • 核心状态显示、来源控制和审批桥接在本机完成。
  • 隐私页说明了读取、写入和网络边界。

安装说明

安装 v0.1.0。

  1. 01 下载并校验

    下载 zip 和同名 .sha256 文件，在同一目录运行 shasum -a 256 -c Vibelsland-Free-0.1.0-macos.zip.sha256。

  2. 02 移动到 Applications

    解压后把应用拖入 Applications。首次打开时按住 Control 点按应用并选择“打开（Open）”。

  3. 03 启用需要的来源

    打开设置页，按需启用 Claude Code、Codex CLI 和 Codex Desktop，并对 Claude Code 或 Codex CLI 安装 hooks。
