安装与信任

下载前最该确认的事。

Vibelsland Free 是免费的吗？

v0.1.0 免费下载，源码公开。下载包通过 GitHub Releases 分发，官网、GitHub Release 文件和 SHA-256 校验值保持一致。

首次打开为什么需要额外确认？

v0.1.0 采用 ad-hoc 签名，因此首次打开时 macOS 可能要求额外确认。请按住 Control 点按应用并选择 Open。

如何确认下载包没有被替换？

下载 zip 和同名 .sha256 文件后，在同一目录运行 shasum -a 256 -c Vibelsland-Free-0.1.0-macos.zip.sha256。官网展示的 SHA-256 与 GitHub Release 文件一致。

使用范围

它会读取什么，支持哪些工具。

支持哪些 AI 编程工具？

v0.1.0 支持 Claude Code、Codex CLI 和 Codex Desktop 的本地状态展示、工具调用摘要和审批请求处理。

核心功能需要互联网连接吗？

不需要。Vibelsland Free 的状态显示、来源控制和本地审批桥接都在本机完成。Claude Code、Codex CLI 和 Codex Desktop 可能有各自的网络连接，但它们独立于本应用。

如何停止或卸载 hooks？

在设置页可以按来源启用、修复或卸载 hooks。应用只会在你选择安装、修复或卸载时修改对应配置。
