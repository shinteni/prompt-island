信任边界

Gatekeeper 与校验结果不是同一件事。

v0.2.0 是 ad-hoc 签名，不是 Developer ID notarized。下载校验通过只能说明 zip 与 GitHub Release 文件一致；如果 spctl 显示 rejected 或 not accepted，那是 notarization 边界，不等于 SHA-256 校验失败。

Vibelsland Free 是独立项目，不隶属于 Anthropic、OpenAI、Claude 或 Codex。

本地数据

应用会把设置和日志保存在本机。

Vibelsland Free 是本地工具。需要反馈问题时，通常只要说明应用版本、macOS 版本、看到的现象和复现步骤。分享截图或日志前，请遮挡账号、个人文件路径、对话内容和其他敏感信息。

应用数据位置 ~/Library/Application Support/VibelslandFree/config.json ~/Library/Logs/VibelslandFree/app.log ~/.vibelsland-free

卸载

完全移除前，先断开本地连接。

  1. 01 断开本地连接

    在应用的设置页或菜单栏执行 Uninstall Hooks。它会移除 Vibelsland Free 为本地状态显示添加的连接项。

  2. 02 退出并删除应用

    从菜单栏退出 >_ - island（官网产品名 Vibelsland Free），然后删除 Applications 中的 >_ - island.app，或删除你手动放入 Applications 的应用副本。

  3. 03 按需删除本地数据

    如果不再需要历史配置和日志，可以删除 ~/Library/Application Support/VibelslandFree、~/Library/Logs/VibelslandFree 和 ~/.vibelsland-free。
