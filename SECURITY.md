# Security Policy

Vibelsland Free is a local-first macOS app. It reads local Claude Code, Codex CLI, and Codex Desktop state so it can display session status, tool activity, and approval requests.

## Reporting Vulnerabilities

Please report security vulnerabilities through GitHub private vulnerability reporting:

https://github.com/shinteni/prompt-island/security/advisories/new

Do not include exploit details, tokens, private paths, prompts, conversation content, or full logs in public GitHub Issues. If private vulnerability reporting is unavailable, open a minimal public issue asking for a private contact path and keep technical details out of the public report.

## Supported Versions

The current public release is `v0.1.0`. Security fixes are handled on `main` first and included in the next public release when applicable.

## What To Include

- A short impact summary.
- Affected version or commit.
- Reproduction steps with sensitive values removed.
- Relevant logs only after redacting account names, private paths, prompts, tokens, and session content.

General bugs and feature requests should use public GitHub Issues after removing sensitive information.
